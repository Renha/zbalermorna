Zbalermorna Font Examiner

This page has contains various featues of the Zbalermorna typeface, and can be used to preview fonts during development. You can drag and drop your font file (any format) to load it here. If you've merged the tools/zbalermorna.fea feature file into your font, you should be able to copy text directly from this page into the metrics window in FontForge to modify things if you see that something is off. Check out the Javascript Zbalermorna IME to test out everything in real-time. You can also use an OS IME to input text as well. All of this page was dynamically converted to the Zbalermorna Unicode space using the Javascript IME engine.
